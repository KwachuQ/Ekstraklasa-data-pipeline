# Data pipeline - Sofascore (Football Stats)

End-to-end data pipeline based on medallion architecture.

# Stack

- Apache Airflow
- dbt-core
- Postgres/Duck DB (warehouse)
- MiniIO (object storage)
- Metabase (BI visualization)
- Docker & docker-compose (enviroment)

# Purpose

To build a pipeline in order to download and prepare football stats for visualization from API Sofascore.
